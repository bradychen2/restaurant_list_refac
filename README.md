# 我的餐廳清單

使用 Node.js and Express 製作的個人餐廳清單，使用者可以點入頁面查看餐廳詳細內容。另外能以關鍵字搜尋符合的餐廳名稱或類型。

## 環境 Environment
Node.js v10.15.0

## 模組 Modules

1. Web Frame: Express ^4.17.1
2. View Engine: express-handlebars ^5.2.0

## 安裝 Installing

1. Clone 專案至本地電腦
2. Terminal 移動至專案資料夾
3. Terminal 輸入 npm install
4. Terminal 輸入 node app.js
